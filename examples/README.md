# 真实案例

以下是底稿系统在真实场景中的使用效果。

## 案例 1：日记系统（内容创作者版）

**使用者**：内容创业者，公众号 10 万粉丝，日更 500+ 天

**底稿内容**：个人风格指纹 + 6 步日记处理流程 + 内容资产库 + 审核规则

**使用方式**：
1. 每天口述 3-5 分钟日记
2. AI 按底稿中的流程整理成文章
3. 整理过程中自动提取故事、洞见、金句
4. 创作者审核通过后存档

**成果**：
- 500+ 天持续产出，AI 整理的文章风格一致
- 积累了 211 条故事、45 条洞见、49 条风格表达
- 换新对话窗口不断档，因为底稿记录了所有状态

**关键底稿配置**：
- 风格部分写了 15+ 条具体的句式特征和禁区
- 流程部分定义了"先审核再存档"的强制规则
- 资产库编号自动接续，跨对话不重复

---

## 案例 2：投资学习（学习系统版）

**使用者**：投资零基础，目标是能独立做个股价值判断

**底稿内容**：学习者状态 + AI 教练角色 + 三阶段路线 + 已掌握框架清单

**使用方式**：
1. 每天 15-30 分钟，看 AI 生成的日报 + 提问
2. AI 给数据，引导推导，不直接给结论
3. 每次学完，AI 更新底稿中的"已掌握框架"
4. 新对话自动从当前水平接续

**成果**（11 天）：
- 从 0 个框架增长到 20+ 个分析框架
- 覆盖宏观（利率/资金流向）、行业（产业链/供需）、公司（财务/护城河）、估值（PE/PEG）
- 能独立对个股做基础价值判断

**关键底稿配置**：
- AI 角色明确写了"数据给我看，让我自己推导"
- 已掌握框架清单每次自动更新，成了活的知识库
- 规矩里写了"分析要完整，不要说一半留一半"
